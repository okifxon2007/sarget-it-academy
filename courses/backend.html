Kurs haqida

Backend dasturlash kursi server tomonida ishlaydigan dasturlarni yaratishni o'rganishni xohlaganlar uchun mo'ljallangan. Kurs davomida siz Node.js, Python, SQL va boshqa zamonaviy texnologiyalarni o'rganasiz va amaliy loyihalar orqali tajriba orttirasiz.

Kurs yakunida siz:

  • Server tomonida ishlaydigan dasturlarni yaratishni
  • Ma'lumotlar bazalari bilan ishlashni
  • API yaratish va boshqarishni
  • Xavfsizlik prinsiplarini
  • Bulutli serverlar bilan ishlashni
  • Portfolio uchun loyihalar yaratishni o'rganasiz

Kimlar uchun
