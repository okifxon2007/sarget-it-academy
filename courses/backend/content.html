Backend dasturlash kursi haqida

Ushbu kursda siz backend dasturlash asoslarini o'rganasiz. Backend dasturlash - bu veb-saytlar va ilovalarning server tomonidagi qismini yaratish jarayoni. Backend dasturchilari ma'lumotlar bazalari, server arxitekturasi, API va boshqa server tomonidagi texnologiyalar bilan ishlaydilar.

Kursda nimalarni o'rganasiz:

  • Node.js va Express.js asoslari
  • Ma'lumotlar bazalari bilan ishlash (MongoDB, MySQL)
  • REST API yaratish
  • Autentifikatsiya va avtorizatsiya
  • Server xavfsizligi
  • Bulutli serverlar bilan ishlash

Kurs davomiyligi:

Kurs 5 moduldan iborat bo'lib, har bir modul bir necha darslarni o'z ichiga oladi. Kursni to'liq o'zlashtirish uchun taxminan 3 oy vaqt talab etiladi.

Kurs uchun talab qilinadigan bilimlar:

  • HTML, CSS va JavaScript asosiy bilimlari
  • Dasturlash asoslari haqida tushuncha
  • Kompyuter savodxonligi
